# Trabalho-Pratico-IBD
Trabalho prático realizado para a disciplina de Introdução a Banco de Dados. Ministrada pelo professor Clodoveu Davis (2025/2). 
O trabalho teve como objetivo realizar a captura de dados que pudessem ter alguma correlação entre si. 
Concentrou-se em analisar os dados de trabalho análogo a escravidão no Brasil e avaliar as suas ligações com a área agropecuária, instrução populacional e nível de pobreza, bem como investigar quais setores estão mais ligados a casos de trabalho forçado.
